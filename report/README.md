## Project Report

### BTech Final Year Major Project
**Title:** AI-Based Fake News Detection System

**Tech Stack**
- Language: Python
- ML Models: TF-IDF + Logistic Regression, BERT
- Libraries: HuggingFace Transformers, PyTorch, Scikit-learn
- API: Google Fact Check Tools API, NewsAPI
- Platform: Google Colab (T4 GPU)

### Abstract
This project implements an AI-based fake news detection 
system using a two-stage pipeline. TF-IDF with Logistic 
Regression serves as the baseline model achieving ~92-94% 
accuracy. BERT (bert-base-uncased) fine-tuned on fake news 
data achieves 98.40% accuracy. An external API verification 
layer using Google Fact Check API and NewsAPI adds real-world 
credibility to predictions.
